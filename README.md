# Cross-Axis Analysis

**Cross-Axis Analysis Framework | 横纵分析法**

通过纵向时间深度 + 横向竞争格局的交汇分析，快速建立对陌生领域的系统性认知。

## 概述

当需要系统性研究一个陌生领域、产品、公司或技术概念时，通过纵向时间线分析 + 横向同期竞争格局对比，快速构建认知框架。

## 使用场景

| 场景 | 版本 | 预期产出 |
|------|------|---------|
| 学术研究（文献综述） | 深度版 | 完整发展脉络 + 竞品对比 |
| 选校/选公司决策 | 深度版 | 生态位分析 + 横纵交汇判断 |
| 投资分析（币/股） | 深度版 | 路径依赖分析 + 未来展望 |
| 快速了解新领域 | 快速版 | 3 个转折点 + 3 个竞品 |

## 安装

```bash
clawhub install cross-axis-analysis
```

## 使用

当遇到需要分析的新领域时，触发此 Skill 即可自动执行横纵分析流程。
